# AI 기반 맞춤형 뉴스레터 큐레이터
> **정보 과잉 시대, 당신에게 꼭 필요한 뉴스만 AI가 요약해 드립니다.**

## 📺 프로젝트 개요 (Overview)
- **프로젝트 명:** AI 큐레이션 서비스
- **주요 타겟:** 정보 습득 시간을 단축하고 싶은 대학생 및 직장인

## 🚀 개발 배경 및 목적 (Background & Goal)
현대 사회의 정보 과잉 문제로 인해 필요한 정보를 선별하는 데 많은 시간이 소요됩니다. 사용자가 관심 있는 키워드의 뉴스나 기술 블로그를 자동으로 수집하고, 생성형 AI(LLM)를 활용해 핵심 내용을 3줄로 요약함으로써 효율적인 정보 소비를 돕는 것을 목적으로 합니다.

## ✨ 주요 핵심 기능 (Main Features)

1. **🔍 스마트 크롤링:** 설정한 관심 키워드 기반 뉴스 실시간 수집
2. **🤖 AI 자동 요약:** GPT API를 활용한 본문 핵심 3줄 요약 제공
3. **📊 트렌드 대시보드:** 오늘 수집된 정보의 핵심 키워드를 시각화
4. **🔔 맞춤형 알림:** 지정된 시간에 요약된 뉴스레터 푸시 알림
5. **⭐ 지식 아카이브:** 중요한 게시글 스크랩 및 사용자 피드백 기반 추천 최적화
